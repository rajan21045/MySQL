## Setup

1. Install [MySQL](https://dev.mysql.com/downloads/mysql/) or use [MySQL Workbench](https://dev.mysql.com/downloads/workbench/).
2. Clone this repository:
   ```bash
   git clone https://github.com/rajan21045/<repo-name>.git
   ```
3. Open any `.sql` file in MySQL Workbench or run via terminal:
   ```bash
   mysql -u root -p < 01-select-and-filtering/queries.sql
   ```
